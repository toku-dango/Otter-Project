# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Otter とは

**画面文脈つきAI相談アシスタント** — ショートカットキー（Ctrl+Shift+Space）または音声で即座に呼び出せる、低摩擦な作業支援アシスタント。

「別画面を開く → コピー → 背景を説明する → 依頼する」という既存AIの起動コストをゼロにする。現在の画面文脈と直前の作業履歴をシステムが瞬時に理解し、ユーザーが短い指示を投げるだけで応答する。

### ポジショニング

自律実行エージェント（Claude Code、Operator等）ではなく「相談UI」。

| 特性 | 内容 |
|---|---|
| 読取専用（Read-Only） | 画面を見るだけ。システムへの書き込み・ファイル操作は行わない |
| 相談特化（Consultation-First） | 提案はするが、実行（ファイル操作等）は絶対にしない |
| 日常特化 | AIを習慣化させる教育的・習慣化目的のUI |

### やること / やらないこと

**やること**
- ショートカット / 音声での即時呼び出し
- 画面理解（メール、PR、スライド等をOCR＋マルチモーダルLLMで解釈）
- 軽量な短期セッション理解（直前の画面要約・会話履歴の保持）
- 短い指示（「短く」「丁寧に」「何か変？」）の解釈と文案出力

**やらないこと**
- コピペやファイル添付の要求
- 常時監視による先回り提案（ユーザー主導のみ）
- システム操作の自律実行（文案提示のみ）

## 技術設計（6要素）

| 要素 | 内容 |
|---|---|
| 画面理解 | OCR、マルチモーダルLLM、UI要素抽出 |
| 状況把握 | 現在画面と直前遷移からの意図推定 |
| 短期記憶 | 直前数分の画面要約、会話履歴、採用/却下履歴 |
| コンテキストエンジニアリング | 画面要約＋履歴＋指示を束ねる最適化 |
| 音声インタラクション | 短い口頭指示の解釈、曖昧さの画面補完 |
| 安全設計 | 保存しないデフォルト、機密マスク、アプリ単位除外 |

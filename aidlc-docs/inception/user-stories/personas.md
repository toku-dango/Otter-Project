## P1: 佐藤 美咲（営業・企画担当）— MVP最優先

| 項目 | 内容 |
|---|---|
| **年齢** | 32歳 |
| **役割** | 営業企画担当 |
| **AIリテラシー** | 中程度（ChatGPTを週数回使う程度） |
| **主な作業環境** | Gmail・PowerPoint・Word・Excel |

**課題:**
- メール文面のトーン調整に毎回時間がかかる
- 提案書作成中にAIを使いたいが、別ウィンドウへの切り替えで集中が途切れる
- 「背景を説明する」手間が大きく、AIを使う気になれない場面が多い

**Goals:**
- 作業フローを止めずに「丁寧に」「端的に」の一言でAIに相談したい
- 今見ている画面をそのまま文脈として使いたい
- コピペ不要で提案結果をすぐ使いたい

**主な利用シーン:** Gmail返信・PowerPoint資料作成・議事録整理・提案書ブラッシュアップ

## P2: 田中 健太（エンジニア）

| 項目 | 内容 |
|---|---|
| **年齢** | 28歳 |
| **役割** | ソフトウェアエンジニア |
| **AIリテラシー** | 高（Claude Code・Copilot等を日常利用） |
| **主な作業環境** | VS Code・GitHub・Terminal・Slack |

**課題:**
- コーディング中の「フロー状態」を維持したまま、PRコメント・コード説明を改善したい
- GitHub PR画面でレビューコメントや説明文を即座に書き換えたい
- Claude Codeは使えるが「ファイル操作なしに相談だけしたい」場面がある

**Goals:**
- コーディング中断なしにテキスト品質を高めたい
- 「この関数の説明を追加して」と画面を見せるだけで完結させたい

**主な利用シーン:** VS Code・GitHub PR・コードレビュー・README作成

## P3: 山田 拓也（PM/PL）

| 項目 | 内容 |
|---|---|
| **年齢** | 35歳 |
| **役割** | プロジェクトマネージャー |
| **AIリテラシー** | 中程度 |
| **主な作業環境** | Notion・Slack・Teams・Excel・メール |

**課題:**
- 多くの関係者へのコミュニケーション文面を毎日大量に書く
- 議事録・要件書・報告書をその場で整えたい

**Goals:**
- 議事録を見ながら関係者向けメールを即座に生成したい
- 複数ツール間でも直前の文脈を引き継いで相談したい

**主な利用シーン:** Notion議事録・Slackメッセージ・Teams会議後フォローアップ

## P4: 鈴木 里奈（コンサルタント）

| 項目 | 内容 |
|---|---|
| **年齢** | 30歳 |
| **役割** | ITコンサルタント |
| **AIリテラシー** | 高 |
| **主な作業環境** | PowerPoint・Word・Confluence・Excel |

**課題:**
- クライアント向け資料の文章表現に高い品質が求められる
- 複数画面で作業しながら資料のブラッシュアップをしたい

- 資料作成フローを止めずに文章品質を高めたい

**主な利用シーン:** PowerPoint資料作成・提案書・報告書・Confluence文書

## P5: 伊藤 誠（管理職）

| 項目 | 内容 |
|---|---|
| **年齢** | 48歳 |
| **役割** | 部長 |
| **AIリテラシー** | 低（AI活用経験ほぼなし） |
| **主な作業環境** | Outlook・Word・Excel |

**課題:**
- AIツールの使い方がわからず、導入コストが高い
- メール・報告書の文面を整えたいが、「プロンプトを書く」発想がない
- 操作が複雑なツールは使わない

**Goals:**
- ショートカット1つで起動し、説明なしで使えるUI
- 「もう少し丁寧に」の一言で完結したい
- 初回セットアップが3ステップ以内で完了すること

**主な利用シーン:** Outlook返信・Word報告書・社内稟議文書

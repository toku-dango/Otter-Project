# NFR Requirements — Unit 2: UI & Configuration

## パフォーマンス要件

| NFR-ID | 要件 | 目標値 | 計測方法 |
|---|---|---|---|
| PERF-U2-01 | ウィジェット表示（show()） | 100ms 以内 | ログのタイムスタンプ差分 |
| PERF-U2-02 | ウィジェット非表示（hide()） | 100ms 以内 | ログのタイムスタンプ差分 |
| PERF-U2-03 | 状態切り替え（set_state()） | 50ms 以内 | 目視確認 |
| PERF-U2-04 | マークダウンレンダリング | 200ms 以内（通常の応答長） | ログのタイムスタンプ差分 |
| PERF-U2-05 | アニメーションフレームレート | 10〜15fps（GIF使用時） | 目視確認 |

---

## 信頼性要件

| NFR-ID | 要件 | 詳細 |
|---|---|---|
| REL-U2-01 | tkinterweb フォールバック | `tkinterweb` のインポート失敗時は CTkTextbox（プレーンテキスト）に自動切替 |
| REL-U2-02 | GIF 読み込み失敗 | GIF ファイルが存在しない・破損している場合はプレースホルダー絵文字に自動切替 |
| REL-U2-03 | config.json 破損 | 読み込み失敗時はデフォルト設定で起動（エラーで落ちない） |
| REL-U2-04 | クリップボードコピー失敗 | `pyperclip` 失敗時はエラーを握りつぶし、ステータスに「コピーに失敗しました」を表示 |

---

## 外観・UX要件

| NFR-ID | 要件 | 詳細 |
|---|---|---|
| UX-U2-01 | カラーテーマ | Windows のシステム設定（ダーク/ライト）に自動追従。`customtkinter.set_appearance_mode("system")` を使用 |
| UX-U2-02 | 常時最前面 | 他のアプリが前面でも FloatingWidget が隠れない |
| UX-U2-03 | フォーカス非奪取 | ウィジェット表示時、元アプリのフォーカスを可能な限り維持する |
| UX-U2-04 | 最小化時のサイズ | 最小化時は高さ 50px（ヘッダーバーのみ）まで縮小 |
| UX-U2-05 | リサイズ下限 | 幅 300px・高さ 200px を下回らない（BR-U2-02） |

---

## セキュリティ要件

| NFR-ID | 要件 | 詳細 |
|---|---|---|
| SEC-U2-01 | config.json | パーミッション 644（所有者読み書き、他読み取り）で作成 |
| SEC-U2-02 | tkinterweb の外部リンク | レンダリング時に外部 URL へのアクセスを無効化（ローカル HTML のみ） |

---

## 保守性要件

| NFR-ID | 要件 | 詳細 |
|---|---|---|
| MAINT-U2-01 | テスト方針 | Unit 2 は UI コンポーネント中心のため自動テストは最小限。ConfigManager・ClipboardService のロジック部分のみ pytest でテスト |
| MAINT-U2-02 | data-testid | 全インタラクティブ要素に `data-testid` 属性を付与（CLAUDE.md 要件） |
| MAINT-U2-03 | GIF差し替え | `assets/` にファイルを置くだけで差し替え可能な設計を維持 |
